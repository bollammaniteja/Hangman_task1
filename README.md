Hangman Game

Project Overview

This project is a simple command-line Hangman game developed using Python. The game selects a random word, and the player has to guess it one letter at a time within a limited number of attempts.

Features

- Random word selection
- User-friendly command-line interface
- Input validation for letter guesses
- Tracks correct and incorrect guesses
- Displays win or lose message

Technologies Used

- Python 3

How to Run

1. Download or clone this repository.
2. Open the project folder in your terminal or command prompt.
3. Run the following command:

python Hangman_task1.py

Learning Outcomes

- Python programming
- Loops and conditional statements
- Lists and strings
- Functions
- Random module
- User input handling
